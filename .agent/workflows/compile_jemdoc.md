description: How to compile the jemdoc website
---
To compile the jemdoc website:
1. Ensure you are in the website root directory: `cd /Users/zxwang/Documents/ShiningSord.github.io`
2. Activate the homepage Python 2 conda environment: `source ~/.zshrc && conda activate homepage`
3. Run the jemdoc compiler: `python jemdoc.py index.jemdoc`
